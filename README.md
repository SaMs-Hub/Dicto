# Dicto

Dicto is an Smart Dictionary developed to generate definitions of the words asked by the User!!

![NewPiccc](https://user-images.githubusercontent.com/76864645/125816926-5b5bfd82-86b4-48e2-b5f4-ab1de859a5da.jpg)


Made with Python, Dicto recognizes even if the User inputs a Wrong word and suggests him similar word
![Pic2](https://user-images.githubusercontent.com/76864645/125817048-7458b92a-7f1c-4cdf-b70c-791554873d80.jpg)


Finally, if the word is not correct, Dicto gives a proper output to the User!
